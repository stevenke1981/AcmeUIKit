Acme UI Kit is an independently authored starter project inspired by the architecture and public design ideas of `longbridge/gpui-component`.

This package does **not** vendor or redistribute that project's source code. It uses GPUI directly and provides a smaller, original component implementation intended as a clean starting point.

Reference project:
- `longbridge/gpui-component` — Apache-2.0
- GPUI / Zed — licenses governed by the upstream repository
- Visual design direction: modern desktop controls and token-based design systems

Before commercial distribution, review all upstream dependency licenses generated by `cargo deny`, `cargo about`, or an equivalent tool.
