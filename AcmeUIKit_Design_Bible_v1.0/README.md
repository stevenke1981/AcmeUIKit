# AcmeUIKit Design Bible v1.0

這是一套可直接放進 AcmeUIKit 專案的 GPUI Desktop Design System 規範包。

## 文件
- `ACME_DESIGN_BIBLE_v1.0.md`：總綱、品牌、視覺方向與完成標準
- `TOKENS.md`：顏色、字體、間距、圓角、陰影、密度、動畫
- `COMPONENT_STANDARDS.md`：核心元件 Anatomy、狀態與 API 規格
- `DESKTOP_PATTERNS.md`：設定、Dashboard、檔案管理器、IDE、SpeakType Cloud
- `ACCESSIBILITY_AND_QA.md`：鍵盤、焦點、對比、視覺回歸與 Release Gate
- `AGENT_IMPLEMENTATION_GUIDE.md`：Codex/OpenCode 任務拆分與驗收規則
- `MIGRATION_PLAN.md`：從目前 AcmeUIKit 遷移至 Visual System v2
- `TOKEN_SCHEMA_EXAMPLE.json`：程式化 token 範例

## 建議位置

```text
docs/design/
```

> Linear 的資訊層級、Zed 的桌面密度、Arc 的精緻互動、Apple Settings 的清楚結構。
